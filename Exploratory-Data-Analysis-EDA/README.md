# Exploratory Data Analysis (EDA) & Visualization

This folder contains Jupyter Notebooks dedicated to understanding the underlying patterns, distributions, and relationships within the dataset using statistical summaries and data visualization techniques.

##  Visualizations Implemented
* **Distribution Analysis:** Utilizing `sns.histplot` with Kernel Density Estimation (KDE) to inspect data skewness and normality.
* **Outlier Detection:** Implementing `sns.boxplot` and `sns.violinplot` to identify data spread, quartiles, and statistical outliers across different categories.
* **Feature Relationships:** Constructing correlation matrices visualized via `sns.heatmap` to detect multicollinearity and feature dependencies.

##  Technologies Used
* **Python 3**
* **Pandas & NumPy:** For data aggregation and statistical calculations.
* **Matplotlib & Seaborn:** For high-quality, professional data visualizations.

##  Objective
The insights derived from this EDA phase guide the feature selection process, help identify anomalies, and lay the foundation for choosing appropriate Machine Learning algorithms.
