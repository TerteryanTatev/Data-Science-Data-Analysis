# Data Cleaning and Preprocessing

This folder contains Python scripts and Jupyter Notebooks focused on the essential steps of data cleaning and preprocessing. Before feeding data into any Machine Learning model, ensuring data quality is crucial.

## Key Operations Implemented
* **Handling Missing Data:** Detecting NaN values and strategically filling them using statistical imputation (Mean/Median for numerical data, Mode for categorical data).
* **Data Consistency:** Removing duplicate records and fixing structural inconsistencies.
* **Type Conversion:** Standardizing data types (e.g., converting object types to categories or floats where appropriate) for optimized memory usage.

## Technologies Used
* **Python 3**
* **Pandas:** For robust data manipulation and dataframe management.
* **NumPy:** For efficient numerical and vector operations.
